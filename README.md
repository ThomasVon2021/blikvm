# Open and cheap DIY and plug-n-play KVM over IP

[![Discord](https://img.shields.io/discord/943534043515977768?color=0&label=chat&logo=discord)](https://discord.gg/9Y374gUF6C)  
BliKVM is an open KVM. It comes in three device types: BliKVM CM4, BliKVM HAT and BliKVM PCIe.
These devices help to manage servers or workstations remotely, regardless of the health of the operating system or whether one is installed.   
The website: [blicube.com](www.blicube.com). Also check out [the documentation](https://wiki.blicube.com/blikvm/) and join to the [Discord Community Chat](https://discord.gg/9Y374gUF6C) for news, questions and support!

## 1.Introduction

This product’s key features include video capture, ATX adapter, USB 3.0, OTG, Gigabit Ethernet, OLED and RTC. The product has a customized metal shell to dissipate heat and provide protection for the circuit board. This is a feature demonstration [video](https://www.youtube.com/watch?v=d7I9l5yG5M8).The product is currently perfectly compatible with  [**v3-hdmi-rpi4-latest.img**](https://files.pikvm.org/images/v3-hdmi-rpi4-latest.img.xz).

## 2.Features

| --------------------------------------------- | ------------------------------------------ |

Here is a brief introduction. For details, please click <a href="https://wiki.blicube.com/blikvm/" target="_blank">Docs</a>
1. **HDMI IN** port with I2S
2. **ATX** controller interface (power on/off, reboot control, PWR and HDD ACT LEDs)
3. **USB3.0 port**  x 2
4. **USB-C OTG** 
5. **Real Time Clock (RTC)**  
6. **Gigabit Ethernet**  
7. **Activity** LED  
8. **Micro SD Card** socket
9. **Power** LED
10. **I2C** display connector
11. **nRPI_BOOT** Jumper
12. **USB-C power input** 
13. PWM **FAN** controller
14. **Serial** console port
15. Support **POE**
